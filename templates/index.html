Sistema de Semaforización Inteligente

Fase I - Modelado y Análisis de Datos para una Ciudad Inteligente

1. Introducción

El presente proyecto forma parte del desarrollo de una Ciudad Inteligente ficticia, cuyo propósito es integrar herramientas tecnológicas y sistemas de información para optimizar la gestión urbana. En esta primera fase se trabajó en el subsistema de semaforización inteligente, orientado a mejorar el flujo vehicular mediante el análisis de datos en tiempo real.

El trabajo consistió en diseñar y construir una base de datos relacional que almacenara información proveniente de sensores, vehículos, ubicaciones y fechas. Posteriormente, los datos fueron cargados mediante procesos ETL (Extract, Transform, Load) utilizando SQL Server Integration Services (SSIS), garantizando su integridad y consistencia.

Finalmente, se elaboró una base de datos multidimensional, basada en el modelo relacional, para facilitar la consulta, análisis y evaluación del comportamiento del tráfico, sentando las bases para la futura toma de decisiones en el contexto de la ciudad inteligente. Además, se realizaron pruebas de rendimiento comparando las consultas ejecutadas en ambos modelos con el fin de analizar la eficiencia de cada estructura.

Contexto: El proyecto se centra en la movilidad de la ciudad de Bogotá, una de las más congestionadas del mundo. El sistema toma datos cada 5 minutos, las 24 horas del día, sobre el flujo vehicular y la velocidad en cada semáforo. Mediante procedimientos almacenados, el tiempo de los semáforos se ajusta dinámicamente para reducir la congestión y mejorar la movilidad urbana.

2. Objetivos

Objetivo General

Desarrollar la primera fase del sistema de información para la Ciudad Inteligente, enfocada en la creación, carga y modelado de una base de datos relacional y multidimensional que soporte el análisis del tráfico vehicular mediante semáforos inteligentes.

Objetivos Específicos

  • Diseñar un modelo relacional que represente las entidades principales del sistema: vehículos, ubicaciones, fechas, semáforos y velocidades.
  • Implementar procesos ETL en SSIS para la carga controlada de los datos hacia la base de datos SQL Server.
  • Crear un modelo multidimensional derivado del modelo relacional, optimizado para análisis y consultas de alto nivel.
  • Realizar consultas comparativas entre ambos modelos y medir los tiempos de respuesta para evaluar el rendimiento.
  • Identificar y documentar errores o dificultades surgidas durante la carga y la ejecución de las consultas, proponiendo posibles soluciones.
